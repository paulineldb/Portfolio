Front-End Developer

Hi! My name is Pauline. I design and code some cool things.

Since childhood, I have been battling between my creative and logical sides. I started playing around with HTML and CSS as a teenager, forgetting about it for a few years as I became a teacher. But I quickly realised I was missing the challenge of code and decided to dive back in, extend my skill set and become a Web Developer!

I am passionate about clean and impactful design, efficient code, and always longing to learn more (might explain the long list of hobbies I have picked up over the years).

The cool things
